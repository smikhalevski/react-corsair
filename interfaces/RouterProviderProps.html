base?: string

The base URL or pathname.

Optional isPendingFallbackForced

isPendingFallbackForced?: boolean

If true then pending fallback is always rendered when a route is being loaded.

If false then the current route is shown when the next route is being loaded. If there's no current route and the next route must be loaded, a pending fallback is rendered.

Default

false

Optional notFoundFallback

notFoundFallback?: ReactNode

Fallback that is rendered if the route that matches the url wasn't matched by any route, or if notFound was called during rendering.
