Optionalcontext

context?: Context

An arbitrary context.

OptionalerrorComponent

errorComponent?: ComponentType

A component that is rendered when an error was thrown during route rendering.

A Router-level errorComponent is used only for root routes. Nested routes must specify their own error components.

Routes without an errorComponent don't have an error boundary.

Inherited from Fallbacks.errorComponent

OptionalloadingComponent

loadingComponent?: ComponentType

A component that is rendered when a lazyComponent or a dataLoader are being loaded. Render a skeleton or a spinner in this component to notify user that a new route is being loaded.

A Router-level loadingComponent is used only for root routes. Child routes must specify their own loading components.

Routes without a loadingComponent suspend a parent route.

Inherited from Fallbacks.loadingComponent

OptionalnotFoundComponent

notFoundComponent?: ComponentType

A component that is rendered if notFound was called during route rendering or if there's no route that matches the location a router was navigated to.

Routes without notFoundComponent propagate notFound to a parent route.

Inherited from Fallbacks.notFoundComponent
