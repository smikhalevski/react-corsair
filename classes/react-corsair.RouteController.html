    error

    • get error(): any

      Returns the error that was thrown during component or data loading, or undefined if status isn't "error".

      Returns any

    renderingStage

    • get renderingStage(): "fallback" | "foreground" | "background" | "discarded"

      Returns the stage at which the controller is currently rendered:

      "foreground"
      The route is visible to the user.
      "background"
      The route is being loaded while a fallback is visible to the user.
      "fallback"
      The route is visible to the user but would be replaced by a route that is being loaded in the background.
      "discarded"
      The route was replaced by another route and isn't visible to the user anymore.

      Returns "fallback" | "foreground" | "background" | "discarded"

    status

    • get status(): "error" | "loading" | "not_found" | "redirect" | "ready"

      Returns the current status of the controller:

      "loading"
      The route is being actively loaded.
      "ready"
      The route component and data were successfully loaded.
      "error"
      The route has thrown an error during rendering or from a data loader.
      "not_found"
      The route was marked as not found.
      "redirect"
      The route has requested a redirect.

      Returns "error" | "loading" | "not_found" | "redirect" | "ready"
